# Checklist Pull Request - SmartCampus

Avant de créer une Pull Request, vérifier :

* [ ] Je suis sur ma branche personnelle.
* [ ] J’ai récupéré la dernière version de main.
* [ ] Ma branche est à jour avec main.
* [ ] Le projet s’ouvre avec http://localhost/SmartCampus/public/login.php.
* [ ] WAMP est vert.
* [ ] La base smartcampus existe dans phpMyAdmin.
* [ ] Aucune erreur PHP n’est affichée.
* [ ] Je n’ai pas modifié les noms des tables.
* [ ] Je n’ai pas modifié les noms des attributs.
* [ ] Je n’ai pas ajouté de fichier inutile.
* [ ] Mon commit correspond à une seule tâche.
* [ ] Mon message de commit est clair.
* [ ] J’ai testé la fonctionnalité localement.
* [ ] J’ai ajouté une entrée dans docs/journal_ia.md si j’ai utilisé l’IA.

## Exemples de bons commits

* Chore: initialisation de la structure SmartCampus
* Docs: ajout de l’organisation de travail du groupe
* Feat: ajout de la connexion MySQL
* Feat: ajout de l’authentification utilisateur
* Feat: ajout du dashboard administrateur
* Feat: affichage de la liste des cours
* Fix: correction du chemin vers header.php
* Fix: correction de la requête des inscriptions
* Style: amélioration du formulaire de connexion

## Exemples de mauvais commits

* update
* test
* final
* projet fini
* tout marche
* gros ajout
